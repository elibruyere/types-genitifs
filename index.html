Les types de génitif en latin

Type Fonction Traduction
Possession Indique le possesseur de / à
Partitif Exprime une partie d’un tout de parmi / d’entre
Objectif Le génitif est l’objet du nom de / envers
Subjectif Le génitif est le sujet du nom de
Qualité Exprime une qualité (avec adjectif) de
Prix / valeur Exprime un prix ou une valeur à tel prix
Crime / accusation Indique une accusation ou une peine de (juridique)
Souvenir / oubli Utilisé avec meminī, obliviscor… de
Matière / contenu Indique la matière ou le contenu de
Avec adjectifs Requis par certains adjectifs de
